### misc - 系统杂项测试
---
1.  **版本信息读取**
    用例文件: `misc/version_info/test_version_read.py`
    描述:   读取设备固件号并验证格式正确性，不同设备读取的位置不同
                Chameleon:读取 /oem/usr/bin/version.txt
                Falcon/Falcon-Air: 读取 /oem/usr/conf/version.txt
            返回version.txt中的内容并且显示在日志区
            如果返回为空则自动测试失败，备注：文件内容异常
    支持设备: Chameleon/Falcon/Falcon-Air
    测试类型: 自动
    优先级：P0
    状态: 启用

---
2.  **LED指示灯检查**
    用例文件: `misc/led_check/test_led_flash.py`
    描述:   人工检查设备LED指示灯状态
            adb shell 发送闪烁命令,不同设备命令不同:
                Falcon/Falcon-Air:mosquitto_pub -h localhost -t BER -m $'\x01'
                Chameleon:mosquitto_pub -h localhost -t "L" -m "$(printf '\x03')"
            人工确认侧灯是否闪烁
    支持设备: Chameleon/Falcon/Falcon-Air
    测试类型: 人工
    优先级：P1
    状态: 启用

---
3.  **开关机检查**
    用例文件: `misc/power_test/power_up_down_test.py`
    描述: 人工检查开关机,无需任何操作直接弹窗询问
    支持设备: Chameleon/Falcon/Falcon-Air
    测试类型: 人工
    优先级：P2
    状态: 启用

---
4.  **蜂鸣器测试检查**
    用例文件: `misc/buzzer_test/buzzer_test.py`
    描述:   adb shell发送指令 mosquitto_pub -h localhost -t "CGA" -m $'\xFE'
            人工点击确认
            休眠3秒再进入下一项测试（恢复状态）
    支持设备: Falcon/Falcon-Air
    测试类型: 人工
    优先级：P1
    状态: 启用

---
5.  **屏幕显示测试**

### stepper_motor_control - 步进电机测试
---
1.  **水平电机测试**
    用例文件: `stepper_motor_control/horizontal_motor_test/horizontal_motor_test.py`
    描述:   adb发送水平电机测试指令,不同设备发送的命令不同：
                Chameleon:echo -ne '\x00\x66\x20\x03\x00\x06' | mosquitto_pub -h localhost -t "A" -s
                Falcon/Falcon-Air:echo -ne '\x2A\x00\x00\x23\x28\x00\x02' | mosquitto_pub -h localhost -t AQR -s
            人工点击确认
            adb发送电机复位命令:
                Chameleon:echo -ne '\x00\x62\x20\x03\x00\x06' | mosquitto_pub -h localhost -t "A" -s
                Falcon/Falcon-Air:echo -ne '\x2B' | mosquitto_pub -h localhost -t AQR -s
    支持设备: Chameleon/Falcon/Falcon-Air
    测试类型: 人工
    优先级：P0
    状态: 启用

---
2.  **俯仰电机测试**
    用例文件: `stepper_motor_control/pitch_motor_test/pitch_motor_test.py`
    描述:   adb发送俯仰电机测试指令,不同设备发送的命令不同：
                Chameleon:echo -ne '\x01\x66\x20\x03\x00\x05' | mosquitto_pub -h localhost -t "A" -s
                Falcon/Falcon-Air:echo -ne '\x00\x01' | mosquitto_pub -h localhost -t AYR -s
            人工点击确认
            adb发送电机复位命令:
                Chameleon:echo -ne '\x01\x62\x20\x03\x00\x05' | mosquitto_pub -h localhost -t "A" -s
                Falcon/Falcon-Air:echo -ne '\x00\x02' | mosquitto_pub -h localhost -t AYR -s
    支持设备: Chameleon/Falcon/Falcon-Air
    测试类型: 人工
    优先级：P0
    状态: 启用
